.. GFF-Toolbox`_

.. image:: https://raw.githubusercontent.com/fmalmeida/gff-toolbox/master/images/GFF_ToolBox.png
  :width: 250px
  :align: left

`GFF-toolbox <https://github.com/fmalmeida/gff-toolbox>`_ is a compendium of useful scripts that enable a quick exploration of GFF files.

This python package is intended to be a collaborative project wich means any one is invited to collaborate by flagging issues, suggesting improvements and help on new implementations. For this, we encourage users to read `Chase's tutorial <https://gist.github.com/Chaser324/ce0505fbed06b947d962>`_ on how to fork and collaborate to public repos.

To date, its main steps are:

.. list-table::
   :widths: 20 80
   :header-rows: 1

   * - Module
     - Description
  
   * - Overview
     - A simple module to get a glimpse of your GFF

   * - Convert
     - A module to convert GFF into different formats

   * - Filter
     - A module to robustly filter GFF based on different patterns, keys, columns, etc. 

   * - Plot
     - A module that enables a quick and robust visualization of your annotation
